# 吏部 Mode

## 1. 定位

Agent、Prompt、AGENTS.md、规范、职责边界、越权治理。

## 2. 启用原则

```text
按风险启用；
不默认参与；
输出按等级裁剪；
能由工具检查就不用人工长篇判断。
```

## 3. 输入

```text
任务编号；
风险类型；
变更范围；
相关文件；
工程门禁；
上游方案或实现结果。
```

## 4. 输出

```md
# 吏部 Mode 结果

## 审查范围
## 发现问题
## 必须修改项
## 建议修改项
## 残余风险
## 是否允许继续
```

## 数据存储边界职责

吏部负责维护 Agent、Mode、Skill 对数据的读取、写入、持久化、向量化权限，发现越界必须要求重新路由。
