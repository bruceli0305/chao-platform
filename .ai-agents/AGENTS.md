# AGENTS.md

## 项目智能体制度

本项目采用“朝”v2.2 local MVP。

任何任务必须先经过任务路由器：

1. 判断 L1 / L2 / L3 / L4。
2. 识别风险类型。
3. 判断是否需要皇帝确认。
4. 判断需要启用哪些角色。
5. 明确允许修改范围和禁止修改范围。
6. 完成必要验证后才能声明完成。

## 禁止项

- 不得未定级直接修改代码。
- 不得未确认 A 级事项直接执行。
- 不得新增未审查依赖。
- 不得修改无关文件。
- 不得无验证宣称完成。
- 不得用 UI 层掩盖后端错误。
- 不得吞异常或返回假成功。
