# 任务史官记录索引

本目录保存“朝”v2.2 local MVP 的任务级 Markdown 记录。

## 记录原则

- PostgreSQL 保存运行状态。
- Markdown 保存人类可读档案。
- Git 保存历史变化。
- L1 / L2 / L3 / L4 都必须生成任务记录。
- L3 / L4 如果进入 NEED_CONFIRMATION，也必须记录阻断原因。

## 当前阶段

- 已支持任务创建。
- 已支持任务定级。
- 已支持 PostgreSQL 落库。
- 已支持 Markdown 双写。
- 尚未自动修改真实代码。
