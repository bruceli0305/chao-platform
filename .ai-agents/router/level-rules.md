# 任务等级规则

## L1

低风险单点任务。

必须同时满足：

- 不改接口
- 不改数据
- 不改权限
- 不改部署
- 不增依赖
- 不碰 Secret
- 不改架构
- 可以轻量验证

## L2

普通功能或普通联动任务。

满足任一即为 L2：

- 新增页面
- 新增普通接口
- 前后端联动
- 多个文件协作
- DTO / 字段 / 状态普通变化
- 需要测试或文档同步

## L3

复杂或高风险任务。

满足任一即最低 L3：

- 数据库变化
- 数据迁移
- 权限模型变化
- 认证鉴权变化
- 生产配置变化
- 部署方式变化
- 第三方服务
- 成本明显增加
- 新技术栈
- 架构边界变化
- Secret 相关
- 无法简单回滚

## L4

战略级工程，不得一次性执行，必须拆成多个 L2 / L3。
